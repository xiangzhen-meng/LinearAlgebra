# 行列式的性质

1) 多线性性（每行/列可以线性加） -> 推论：可以对某一行提出公因子
2) 交错性（对换行/列，行列式反号） -> 推论：两行相等或成比例，则行列式为 0
3) 单位性（单位阵行列式为 1）

其中，多线性性的相当于分配律（证明即是用分配律证的）
交错性跟对换的性质类似
单位性可以看作是定义

这三条性质的组合等价于行列式的完全展开式定义，这即是 **行列式的公理化定义** 

证明思路：
构造行向量 $e_{1},e_{2},\dots,e_{n}$ ，使得 $e_{i} = \{0, 0, \dots,1,0,\dots,0\}$ ，1 的位置为第 i 个元素
对每一个行向量，分解为 $a_{i,j_{i}}\cdot e_{i}$ 之和，并提出 $a_{i,j_{i}}$
