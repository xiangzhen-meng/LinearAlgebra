# 距离与范数

### 距离

>[!note] 距离的定义
>距离是建立在点集上的一种结构，建立了这种结构的点集称为 **度量空间**，这种结构满足以下三条：
>1. 正定性：$d(x, y)\geq 0$ 且 $d(x, y)=0 \iff x=y$ 
>2. 对称性：$d(x, y)=d(y, x)$ 
>3. 三角形不等式：$d(x, y)\leq d(x, z)+d(y, z)$ 

### 范数

>[!note] 范数的定义
>对于线性空间 $X^n\cong \mathbb{F}^n$，函数 $\lVert \cdot \rVert$ 称范数，若满足以下三条：
>1. 正定性：$\lVert x \rVert\geq 0$ 且 $\lVert x \rVert = 0 \iff x=0$
>2. 齐次性：$\lVert \lambda x \rVert = \lvert \lambda \rvert\lVert x \rVert$ 
>3. 三角形不等式：$\lVert x+y \rVert\leq \lVert x \rVert +\lVert y \rVert$ 

这样的范数必然能够诱导出一种距离，这个距离可以按如下方式自然的定义：
$$
d(x, y) = \lVert x-y \rVert 
$$
这是良定义，因为这满足距离的三条性质，验证是显然的。
